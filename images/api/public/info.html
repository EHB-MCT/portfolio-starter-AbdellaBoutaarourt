Documentation REST Api Abdella Boutaarourt

/ = Documentation


GET /users = Show all users

GET /users/:id = Show one specific user

POST /users/register = Post a user

POST /users/login = Checks if user send the right data

DEL /users/:id = Delete an user

PUT /users/:id = Update email, name or password of user


GET /animes = Show all animes

GET /animes/:id = Show one specific anime

GET /animes/saved/:id = Show all animes saved by a user

POST /animes = Post a anime

DEL /animes/:id = Delete a anime
